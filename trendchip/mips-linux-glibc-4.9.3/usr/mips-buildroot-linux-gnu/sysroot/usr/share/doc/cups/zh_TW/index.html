關於 CUPS

CUPS 提供以 UNIX® 為基礎的作業系統 一層具有可攜性的列印介面。它是由 Apple Inc. 為了推動一個標準的列印解決方案所開發與維護。CUPS 是目前用於 MacOS® X 與大部份 Linux® 發行版本的標準列印系統。

CUPS 使用 Internet Printing Protocol ("IPP") 做為 管理列印工作與佇列的基礎，並且加入瀏覽網路印表機和基於 PostScript Printer Description ("PPD") 的列印選項， 以支援真實世界中的列印。
